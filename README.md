# To-Do App

A simple and interactive To-Do application built with **vanilla JavaScript, HTML, and CSS**.  
This project demonstrates fundamental **front-end development skills**, including DOM manipulation, event handling, and persistent state using `localStorage`.

## Features

- Create, edit, and delete tasks
- Assign task priority: High, Medium, Low
- Mark tasks as completed
- Optional due date, description, email, and phone number for each task
- Move tasks up or down within their priority column
- Persistent storage using `localStorage`
- Responsive layout for desktop and mobile devices
- Professional code documentation using **JSDoc-style comments**

## Technologies

- HTML5
- CSS3 (Flexbox layout)
- JavaScript (ES6+)
- LocalStorage for persistent state
- FontAwesome icons for UI elements

## Project Structure

- `index.html` — Main HTML structure  
- `style.css` — Styles and responsive design  
- `js/storage.js` — Functions for managing tasks in localStorage  
- `js/script.js` — DOM manipulation, task creation, and event handling  
- `vendor/` — Third-party assets (FontAwesome)

## How to Run

1. Clone the repository:
2. git clone https://github.com/pjsm-it/todo-app
3.  Open `index.html` in your browser.  
4. Start adding and managing tasks immediately — no build tools or server required.

## Notes

- All JavaScript code is documented using **JSDoc-style comments**, demonstrating best practices for readability and maintainability.  
- This project is intended as an **entry-level portfolio piece**, showcasing core front-end skills without relying on frameworks or libraries.
